    • showPreviewOverlay(
          group: DockviewGroupPanel,
          position: Position,
      ): DockviewIDisposable

      Render the drop-preview overlay on a group at position — the same overlay a mouse drag shows — without a live drag. Returns a disposable that clears it. Used by keyboard docking so keyboard and mouse previews are identical (single source of truth). Commit the move via the public api.moveGroupOrPanel({ to: { group, position } }).
