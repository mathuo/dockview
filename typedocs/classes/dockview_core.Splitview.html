Private layoutViews

  • layoutViews(): void

  • Margin explain:

    For n views in a splitview there will be n-1 margins m.

    To fit the margins each view must reduce in size by (m * (n - 1)) / n.

    For each view i the offet must be adjusted by m * i/(n - 1).
