When there is nothing else to display.

When Dockview is empty or a group has no panels (an empty group) you can render some fallback
content which is referred to as a `watermark`. Both are controlled through the same provided component.

## Empty state behaviour

By default, when Dockview has no panels it renders a watermark. You can change this with `noPanelsOverlay`:

- `'watermark'` (default): renders your `watermarkComponent` when empty
- `'emptyGroup'`: keeps an empty group visible instead

<FrameworkSpecific framework="React">
    ```tsx
    <DockviewReact
        noPanelsOverlay="emptyGroup"
        onReady={onReady}
        components={components}
    />
    ```
</FrameworkSpecific>

<FrameworkSpecific framework='Vue'>
```html
<dockview-vue
    noPanelsOverlay="emptyGroup"
    :components="components"
    @ready="onReady">
</dockview-vue>
```
</FrameworkSpecific>

<FrameworkSpecific framework='Angular'>
```html
<dv-dockview
    noPanelsOverlay="emptyGroup"
    [components]="components"
    (ready)="onReady($event)">
</dv-dockview>
```
</FrameworkSpecific>

<FrameworkSpecific framework='JavaScript'>
```ts
const api = createDockview(element, {
    noPanelsOverlay: 'emptyGroup',
    createComponent: (options) => new MyPanel(),
});
```
</FrameworkSpecific>
