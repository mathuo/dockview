Locking the whole component prevents users from resizing groups with the drag handles between them. Combine it with disabling drag and drop for a fully static, read-only layout.

:::info
You may want to combine this with `disableDnd={true}` to provide a locked grid with no drag and drop functionality. See [Disable drag and drop](/docs/core/dnd/disable) for more.
:::

Locking the component prevents the resizing of components using the drag handles between panels.
