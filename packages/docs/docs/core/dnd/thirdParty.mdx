---
title: 'Third-party libraries'
sidebar_position: 2
description: How to integrate Dockview with third-party drag and drop libraries.
---

All third-party drag and drop libraries should work as expected.

Dockview fires and intercepts drag and drop events extensively, however it is intended that the user has ultimate
control over all events.

Dockview should not change the behaviour of any third-party drag and drop libraries.
If you feel that Dockview is causing the behaviour of any third-party drag and drop libraries to change please
raise an issue.

## See also

- [Drag and drop](/docs/core/dnd/dragAndDrop): Dockview's own drag and drop and the events it exposes.
- [External drag and drop events](/docs/core/dnd/external): intercept drops entering the layout from outside.
